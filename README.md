# Bitcoin (BCH) vending machine

*Bitcoin Vending Machine is a simple software that will turn your micro-computer into a fully functional vending machine.*

## Software

- Python (Flask)
- Siema JavaScript Slider

## Hardware

- Raspberry Pi 3
- LCD 3.5inch RPi display
    - 480x320 pixels
